# 🚀 Quick Start Guide

Get started with OmniBlocks in minutes! This guide covers the basics to help you begin your coding journey.

## 🎬 First Launch

1. **Open OmniBlocks**: Visit [https://omniblocks.github.io](https://omniblocks.github.io) or launch your installed PWA
2. **Choose your experience**:
   - **New Project**: Start from scratch
   - **Load Project**: Import existing `.sb3` files
   - **Examples**: Explore sample projects

## 🧩 Basic Interface Tour

### Main Components

1. **Stage Area**: Where your project runs and visual output appears
2. **Blocks Palette**: Drag-and-drop blocks for programming
3. **Code Area**: Where you assemble your blocks
4. **Sprite List**: Manage your sprites and backdrops
5. **Toolbar**: File operations, settings, and tools

### Key Features

- **Turbo Mode**: Faster project execution (toggle in settings)
- **Addons**: Enable/disable features in settings
- **Themes**: Customize the interface appearance
- **Music Editor**: Create custom music tracks

## 🎯 Your First Project

### Step 1: Create a Sprite

1. Click the **"Choose a Sprite"** button (cat icon)
2. Select **"Paint"** to create a custom sprite
3. Draw your character or choose from the library

### Step 2: Add Movement

1. From the **Motion** category, drag:
   - `when green flag clicked` (Events)
   - `forever` (Control)
   - `move 10 steps` (Motion)
   - `if on edge, bounce` (Motion)

2. Assemble them like this:
   ```
   when green flag clicked
   forever
     move 10 steps
     if on edge, bounce
   ```

### Step 3: Add Interaction

1. From the **Events** category, add:
   - `when this sprite clicked`
   - `say Hello! for 2 seconds` (Looks)

### Step 4: Test Your Project

1. Click the **Green Flag** to start
2. Click your sprite to see it respond
3. Watch your sprite move and bounce

## 🎨 Customizing Your Experience

### Enabling Addons

1. Click the **Settings** gear icon
2. Navigate to **Addons**
3. Enable useful addons like:
   - **Editor Devtools**: Advanced editing features
   - **Debugger**: Step-through debugging
   - **Custom Themes**: Additional visual themes

### Changing Themes

1. Click the **Theme** button in the toolbar
2. Choose from available themes:
   - **Aqua** (default OmniBlocks theme)
   - **Blue**
   - **Rainbow**
   - **Red**
   - **Purple**

## 💾 Saving and Sharing

### Saving Projects

1. Click **File** → **Save to your computer**
2. Choose a location and filename
3. Your project saves as a `.sb3` file (compatible with Scratch)

### Loading Projects

1. Click **File** → **Load from your computer**
2. Select your `.sb3` file
3. Your project loads with all sprites, code, and assets

### Sharing Online

1. Click **File** → **Save to cloud** (if available)
2. Share the project link with others
3. Collaborate using cloud features

## 🎵 Using the Music Editor

1. Click the **Music Editor** button in the toolbar
2. Create melodies using the piano roll interface
3. Export your music for use in projects
4. Import music files for advanced editing
