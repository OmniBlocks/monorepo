This branch will add support for the installed PWA of OmniBlocks to be able to directly open files like other native programs but entirely as a PWA. Unfortunately, this will only work on Chromium-based desktop browsers, as the file_handlers API does _not_ exist on Firefox, Safari, or mobile browsers.
